### Demo 1 ###

Hello! Welcome to the ScanFold3 demo/tutorial. This demo assumes ScanFold3 has been installed and the conda/mamba environment has been set up. If this isn't the case, please see the installation instructions on github and the user manual.

ScanFold3 requries a number of libraries to run. These are handled through micromamba, a package manager that creates an environment containing a number of libraries. 

If you are following these instructions on the server, you will need to start an interactive session unless you are already in one. This can be done with the following command:
srun --time=01:00:00 --nodes=1 --cpus-per-task=8 --pty /usr/bin/bash

###

1. If the environment isn't active, activate it. You will know the environment is active if its name appears in parentheses next to the command line, for example:
    (ScanFold3) [ecoppen@nova18-wide-4 build]

    In order to activate the environment, use "micromamba activate":
    micromamba activate /lustre/hdd/LAS/wmoss-lab/programs/envs/ScanFold3

    You should now see (ScanFold3) to the left of the command line. 

2. In order to run ScanFold3, you will need a fasta file. The HIV1 reference genome has been provided here. ScanFold is run using the ScanFold.py script. Optional arguments exist, but now we will just run it:
    python /lustre/hdd/LAS/wmoss-lab/scripts/ScanFold3/ScanFold.py "HIV.fasta"

    This will create a number of files when it finishes running, and will print some debug information to the console while running. 

3. ScanFold3 can also be run using a slurm script. To do this, use the following command:
sbatch SF3.sh

    The SF3.sh script can be used as a template for your own scripts. Depending on the size of your sequence you may need to increase the time and/or memory limits. Information on using slurm can be found on its website:
    https://slurm.schedmd.com/documentation.html

4 (optional). Using HIV.fasta as the genome, try opening up the files under igv_files in the Interactive Genomics Viewer program and comparing them to known annotations in HIV.gff
